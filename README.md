![GalaxEyes banner](banner.png)
**GalaxEyes** is a tool for Super Mario Galaxy 2 that optimizes and checks for issues in your mod. 

It features:
- Removing vanilla files from your mod
- Merging identical collision codes
- AST Optimizer
- ...and more!
